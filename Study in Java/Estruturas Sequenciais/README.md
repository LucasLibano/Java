# Estruturas Sequenciais

Neste repositório estou registrando meus exercícios sobre **estruturas sequenciais**.

Este material faz parte do curso **Java Completo**, ministrado pelo Prof. Dr. Nelio Alves através do portal [Educando Web] ministrado na Udemy.

Os exercícios foram obtidos das aulas do Professor Dr. Nelio Alves e foram resolvidos por mim para estudo.

- Processamento de informações
- Saída formatada
